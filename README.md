# theater_models

## Задание
Cпроектировать pydantic-модели для театра: зритель, актер, постановка и представление

## Запуск простого примера работы
```python -m venv venv```

```source venv/bin/activate```

```pip install requirements.txt```

```python demo.py```

## Запуск тестирования
```pytest```

## Созданные модели

### Модель зрителя [Viewer](src/models/viewer.py)

| Название        | Обязательное значение | Значение по умолчанию | Описание                 |
|-----------------|-----------------------|-----------------------|--------------------------|
| id              | нет                   | counter (start=1)     | уникальный идентификатор |
| name            | да                    | -                     | имя зрителя              |
| surname         | да                    | -                     | фамилия зрителя          |
| email_address   | да                    | -                     | email-адрес              |
| ticket_number   | да                    | -                     | номер билета             |
| phone_number    | нет                   | ""                    | номер телефона           |

#### Конфигурация
```str_strip_whitespace=True```
